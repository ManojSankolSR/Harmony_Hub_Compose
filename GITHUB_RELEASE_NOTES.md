# Release v1.0.0 - Initial Release 🚀

We are thrilled to announce the first official release of **Harmony Hub**, a feature-rich, modern music streaming and management application built entirely with **Jetpack Compose**. 

Harmony Hub aims to provide a seamless and beautiful music experience, combining online streaming with powerful local library management.

## ✨ Key Features
* **Music Streaming & Discovery**: Search and stream a vast library of songs, albums, and artists powered by high-quality APIs.
* **Modern Music Player**: A high-performance player with Media3 (ExoPlayer) integration, featuring beautiful expanded controls, fluid animations, and high-fidelity playback.
* **Personalized Library**: Manage your "Liked" songs, create custom playlists, and organize your music just the way you like it.
* **Artist & Album Exploration**: Deep dive into artist profiles, view their full discography, and explore complete albums with ease.
* **Offline Support**: Download your favorite tracks and manage your local music collection for offline listening.
* **Smart Search**: Quickly find what you're looking for with an integrated, responsive search system.
* **Radio**: Explore curated radio stations for non-stop music discovery.
* **Modern UI/UX**: A stunning Material 3 interface featuring dynamic theming (Palette support) and a smooth, responsive design.

## 🛠️ Technical Highlights
* **UI Framework**: Built with **Jetpack Compose** for a modern, declarative UI.
* **Media Engine**: Powered by **Android Media3 (ExoPlayer)** for robust audio playback and session management.
* **Local Storage**: Uses **Room Database** for efficient caching and local data persistence.
* **Networking**: **Retrofit** and **OkHttp** for optimized API communication and data fetching.
* **Image Loading**: High-performance image loading with **Coil**.
* **Architecture**: Follows clean architecture principles with a focus on modularity and testability.

## 📦 How to Install
1. Download the `app-release.apk` attached to the GitHub release.
2. Open the file on your Android device (Android 7.0+ recommended).
3. If prompted, allow "Installation from Unknown Sources" in your device settings.
4. Launch **Harmony Hub** and start your musical journey!
